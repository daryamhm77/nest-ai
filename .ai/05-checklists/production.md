# Production Readiness Checklist
- [ ] SLOs defined; alerts wired to burn rate.
- [ ] Backups + restore tested; DR plan exists.
- [ ] Rate limiting + abuse protection on public routes.
- [ ] Audit logging for sensitive actions.
- [ ] Resource limits + autoscaling configured.
- [ ] Runbooks linked from alerts.
- [ ] Data privacy/compliance reviewed (`PROJECT.md` constraints).
