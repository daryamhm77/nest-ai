# Roadmap

> Sequenced, not dated. Each phase ships something usable.

## Phase 0 — Foundation
Infra, auth (JWT + refresh + RBAC), audit log, CI/CD, observability baseline.

## Phase 1 — Core domain
The first bounded context end-to-end with full tests.

## Phase 2 — Hardening
Rate limiting, caching, performance budgets, security review (`09-review/security-review.md`).

## Phase 3 — Scale
Read models / CQRS where measured, async messaging, horizontal scaling.

> Definition of phase done: all `05-checklists/feature.md` items green for every shipped feature.
